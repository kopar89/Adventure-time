# Adventure-time

Adventure Time — это консольная RPG игра, написанная на языке Go. Игрок управляет героем, перемещается по случайно генерируемой карте с разными типами ландшафта (трава, лес, горы, вода) и сражается с врагами.

Основные особенности:

Случайно генерируемая карта: каждый раз игра уникальна.

Персонажи с уникальными способностями: каждый герой имеет свои урон, здоровье, ману и спецудары.

Сражения с врагами: пошаговые бои с системой урона и здоровья.

Полиморфизм и композиция в коде: использованы интерфейсы и встроенные структуры Go для гибкости и расширяемости.

Возможность расширения: легко добавлять новых героев, врагов и типы спецударов.

Как играть:

Запустить проект: go run main_games.go.

Следовать инструкциям диалогового интерфейса.

Передвигаться по карте с помощью команд и сражаться с врагами, встречающимися на пути.
